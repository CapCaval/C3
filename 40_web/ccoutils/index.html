Welcome to ccOutils

ccOutils is a Java library which provides you either extra features or simplify the use of existing ones. It is made of several small utilities classes which provides functions such as CLI, Factory, Corba servant, pair etc..

The main classes are :

  • Application This abstraction provides tools to configure and to launch your own application. Auto-documentation is also available.

  • Console Console line can be useful either to use your code inside scripts.

  • Factory Provide 2 kind of abstract Factory, one very generic with less code to write and another more powerful by letting you writing your own factory interface and implementation. Check out your needs and choose one of this

  • Corba Servant, At first Corba is a bit difficult to get in from its rich features. ccTools will help you by simplifying servant creation.

Getting started

The easiest way is to download the ccOutils_samples.zip. Unzip it in any directory and from Eclipse, select the menu "file/import.../General/Existing Projects into Workspace" then press "next" and "Browse" to select the sample directory. If you are interested in Factory there is also an article on it.
